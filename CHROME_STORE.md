# Информация для Chrome Web Store

## Описание (краткое)
Вставляйте зачеркнутый текст в посты Facebook одним нажатием клавиш. Простое и удобное расширение для форматирования текста.

## Описание (полное)
Facebook Зачеркнутый Текст - это простое расширение для Chrome, которое позволяет быстро вставлять зачеркнутый текст в посты Facebook.

**Основные возможности:**
- Быстрая вставка зачеркнутого текста через горячие клавиши (Ctrl+Shift+S)
- Удобный popup интерфейс
- Автоматическое форматирование текста с использованием Unicode символов
- Работает на всех страницах Facebook

**Как использовать:**
1. Откройте Facebook и начните создавать пост
2. Нажмите Ctrl+Shift+S (или Cmd+Shift+S на Mac)
3. Введите текст в появившемся окне
4. Текст автоматически вставится зачеркнутым

Или используйте popup расширения - просто кликните на иконку и введите текст.

**Особенности:**
- Не требует дополнительных разрешений
- Не собирает личные данные
- Работает только на страницах Facebook
- Легкий и быстрый

Идеально подходит для создания интересных постов с зачеркнутым текстом на Facebook.
