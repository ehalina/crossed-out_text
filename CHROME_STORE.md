# Информация для Chrome Web Store

## Описание (краткое)
Вставляйте зачеркнутый текст в посты Facebook одним нажатием клавиш. Безопасная вставка через Unicode символ U+0336 без скриптов. Простое и удобное расширение для форматирования текста.

## Описание (полное)
Facebook Зачеркнутый Текст - это простое расширение для Chrome, которое позволяет быстро вставлять зачеркнутый текст в посты Facebook.

**Основные возможности:**
- Быстрая вставка зачеркнутого текста через горячие клавиши (Ctrl+Shift+S / Cmd+Shift+S на Mac)
- Удобный popup интерфейс с полем ввода
- **Безопасная вставка через Unicode символ U+0336 без использования скриптов** - текст форматируется на уровне символов, а не через JavaScript
- Работает на всех страницах Facebook
- Умное определение поля ввода (работает в окнах создания постов, комментариях и других местах)
- Предотвращение нежелательной прокрутки страницы при вставке текста

**Как использовать:**
1. Откройте Facebook и начните создавать пост (или откройте окно новой записи в профиле)
2. Нажмите Ctrl+Shift+S (или Cmd+Shift+S на Mac)
3. Введите текст в появившемся окне
4. Нажмите Enter или кнопку "Вставить"
5. Текст автоматически вставится зачеркнутым в правильное поле ввода

Или используйте popup расширения - просто кликните на иконку расширения и введите текст.

**Особенности:**
- **Безопасная технология:** Использует Unicode символ U+0336 (Combining Long Stroke Overlay) для зачеркивания - никаких скриптов или модификаций DOM
- Не требует дополнительных разрешений
- Не собирает личные данные
- Работает только на страницах Facebook
- Легкий и быстрый
- Стабильная работа в различных контекстах Facebook (посты, комментарии, окна создания записей)

Идеально подходит для создания интересных постов с зачеркнутым текстом на Facebook.

## Скриншоты
Рекомендуется загрузить минимум 1 скриншот, лучше 3-5:

## Privacy Policy
Расширение не собирает, не хранит и не передает никакие личные данные пользователя. 
Все операции выполняются локально в браузере.

Полная политика конфиденциальности доступна в файле privacy-policy.html. 
Рекомендуется загрузить этот файл на GitHub Pages или другой публичный хостинг и указать URL в настройках расширения в Chrome Web Store.
