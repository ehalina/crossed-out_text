# Распространение расширения

## Способ 1: Упаковка в ZIP (рекомендуется)

### Автоматическая упаковка (через скрипт)

1. **Запустите скрипт упаковки:**
   ```bash
   ./package.sh
   ```
   Или на Windows:
   ```bash
   bash package.sh
   ```

2. **Скрипт автоматически:**
   - Соберет все необходимые файлы
   - Создаст ZIP архив `facebook-strikethrough.zip`
   - Покажет инструкции

### Ручная упаковка

1. **Убедитесь, что все файлы на месте:**
   - manifest.json
   - popup.html
   - popup.js
   - content.js
   - background.js
   - styles.css
   - icon16.png, icon48.png, icon128.png

2. **Создайте ZIP архив:**
   - Выберите все файлы расширения (кроме README.md, DISTRIBUTION.md, generate-icons.html, package.sh)
   - Создайте ZIP архив с именем `facebook-strikethrough.zip`
   - **ВАЖНО:** Не включайте папку в архив, только файлы напрямую

3. **Распространение:**
   - Отправьте ZIP файл пользователям
   - Они должны распаковать его и загрузить как распакованное расширение

## Способ 2: Упаковка через Chrome (для опытных пользователей)

1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Упаковать расширение"
4. Выберите папку с расширением
5. Chrome создаст .crx файл и .pem ключ
6. Распространяйте .crx файл (но пользователи должны включить режим разработчика для установки)

## Способ 3: Chrome Web Store (официальный способ)

1. Создайте ZIP архив (как в способе 1)
2. Зайдите на https://chrome.google.com/webstore/devconsole
3. Создайте новый элемент
4. Загрузите ZIP файл
5. Заполните описание, скриншоты и т.д.
6. Оплатите единоразовый взнос $5 (если еще не платили)
7. Опубликуйте расширение

## Инструкция для пользователей (установка из ZIP)

1. Скачайте `facebook-strikethrough.zip`
2. Распакуйте архив в любую папку
3. Откройте Chrome и перейдите в `chrome://extensions/`
4. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
5. Нажмите "Загрузить распакованное расширение" (Load unpacked)
6. Выберите распакованную папку
7. Готово! Расширение установлено

## Что включить в ZIP

✅ **Включить:**
- manifest.json
- popup.html
- popup.js
- content.js
- background.js
- styles.css
- icon16.png
- icon48.png
- icon128.png

❌ **НЕ включать:**
- README.md
- DISTRIBUTION.md
- generate-icons.html
- .git папки
- node_modules (если есть)
