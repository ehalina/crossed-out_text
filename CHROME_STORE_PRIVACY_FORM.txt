CHROME WEB STORE - PRIVACY FORM TEXT
=====================================

SINGLE PURPOSE (Единая цель расширения)
----------------------------------------
This extension has a single, narrow purpose: to insert strikethrough text into Facebook posts using Unicode character U+0336 (Combining Long Stroke Overlay). The extension provides a quick way to format text with strikethrough effect in Facebook's post creation interface through keyboard shortcuts or popup interface. All functionality is focused solely on this text formatting feature for Facebook posts.

DETAILED PRIVACY INFORMATION
------------------------------

Data Collection:
- This extension does NOT collect, store, or transmit any user data
- All operations are performed locally in the user's browser
- No data is sent to external servers
- No analytics or tracking is implemented

Permissions Used:
- activeTab: Required to work on the active Facebook tab
- scripting: Required to insert text into Facebook's contenteditable fields
- host_permissions (facebook.com only): Required to access Facebook pages where the extension operates

Technology Used:
- The extension uses Unicode character U+0336 (Combining Long Stroke Overlay) for strikethrough formatting
- Text insertion is done through standard browser APIs (execCommand, DOM manipulation)
- No external scripts or libraries are loaded
- No network requests are made

User Data:
- Text entered by the user is processed only locally
- Text is inserted directly into Facebook's input fields
- No text is stored, logged, or transmitted
- No user identification or tracking occurs

Third-Party Services:
- This extension does not use any third-party services
- No data is shared with third parties
- No external APIs are called

Storage:
- This extension does not use browser storage (localStorage, chrome.storage, etc.)
- No persistent data is saved
- All operations are performed in memory only
